# Term3d

## Description

A 3D rendering program in the terminal.

## Requirement

- GCC or Clang
- GNU Make

## Usage

```
git clone <https://github.com/ryo-manba/term3d>
cd term3d && make
```

An executable named term3d will be created.

```
./term3d [FILE_PATH]
```
